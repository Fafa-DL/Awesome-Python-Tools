- `init_video_path`为原始待合并的视频文件夹路径，注意请按合并顺序在`文件名起始位置`给出`数字索引`，否则容易乱序，如
```
001 a.mp4
002 b.mp4
003 c.mp4
```
- `ts_video_path` 为ts格式视频文件夹路径，若从原格式直接转，容易出现间隙过长等问题，统一从ts中转
